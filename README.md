### Getting Started

1.  Clone the repository:
    ```
    git clone https://github.com/Karthikanegouni/react-spa-portfolio-template.git
    ```
2.  Install dependencies:
    ```
    npm install
    ```
3.  Run the app:
    ```
    npm run dev
    ```
